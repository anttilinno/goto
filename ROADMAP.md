# Roadmap

Planned features and improvements for goto.

## Planned

| Feature | Priority | Description |
|---------|----------|-------------|
| Bookmark groups | Medium | Group aliases into collections for easier management |
| Fuzzy search navigation | High | Navigate using partial/fuzzy alias matching |
| Shell history integration | Low | Learn from cd commands in shell history |
| Remote path support | Low | Support SSH paths like `user@host:/path` |
| Alias notes | Medium | Add notes/comments to aliases |

## Completed

| Feature | Version | Description |
|---------|---------|-------------|
| Tags support | 1.3.0 | Tag aliases for organization and filtering |
