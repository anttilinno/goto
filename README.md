# goto

A Go implementation of [goto](https://github.com/iridakos/goto) - a shell utility for navigating to aliased directories with autocomplete support.

## Installation

```bash
mise run build
```

## Usage

Source the appropriate shell script for your shell:

- Bash: `source shell/goto.bash`
- Zsh: `source shell/goto.zsh`
- Fish: `source shell/goto.fish`
